绩效管理

功能概述

绩效统计 就是工作量统计。按照人员分别统计委托单、样品、检测任务和检验报告的次数，以柱状图的方式展现。

操作说明

点击{绩效管理}-->{绩效统计}菜单，弹出绩效统计页面，如下图所示：

在绩效统计页面中，鼠标悬停在柱状图中，可显示当前柱形代表的员工某类工作量。
