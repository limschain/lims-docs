设备管理

设备列表

功能概述

设备列表罗列了本质检机构全部登记的设备信息。

操作包括登记设备信息、删除设备信息和编辑设备信息。操作页面如下图所示：

操作说明

  • 登记设备信息 点击【添加设备】，弹出仪器信息编辑页面，执行仪器信息编辑操作

  • 编辑设备信息 点击【编辑】，弹出仪器信息编辑页面，执行仪器信息编辑操作

  • 删除设备信息 点击【删除】，弹出删除确认对话框，点击【确认】完成删除。
