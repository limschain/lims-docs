样品管理

样品管理 就是对样品在样品室和制样室的管理，包括制样、领样和打印样品标签。

样品列表

功能概述

样品列表就是委托受理样品信息的列表，一条记录就是一个委托样品。

操作说明

点击{实验室信息管理}-->{样品管理}-->{样品列表}菜单，弹出样品列表页面,如下图所示：

(1)在样品列表中可对样品执行批量制样完成和批量领取样品和批量打印样品标签操作。具体操作如下：

  • 批量制样完成 批量制样操作，可以一次操作多条记录，提高效率。操作步骤如下表所示：

第一步 选择样品 选择需要制样的样品

第二步 选择样品 点击【批量】，执打印样品标签操作。

  • 批量领取样品 批量领样操作，可以一次操作多条记录，提高效率。操作步骤如下表所示：

第一步 选择样品 选择需要领样的样品

第二步 选择样品 点击【批量领样】，执行领样操作。

  • 批量打印样品标签 操作步骤如下表所示：

第一步 选择样品 选择需要打印的样品

第二步 选择样品 点击【批量打印】，执行打印样品标签操作。

(2)在操作列中，可以样品为单位 制样完成、 领取样品、 打印样品标签。

  • 制样完成 人员制样任务分配后，制样人员执行制样任务。操作步骤如下：

第一步 点击当前样品右端【操作】-->【制备完成】，弹出制样任务完成提醒对话框，如下图所示：

第二步 点击【对，已完成】弹出，如下图所示：

第三步 点击【OK】 完成制样操作。

  • 领取样品 需要制样的样品完成制样后，由领样人员向制样员领取样品；不需要制样的，检测人员任务分配后，由领样人员向样品管理员领取样品。

    操作步骤如下： 点击当前样品右端【操作】-->【领取样品】，弹出领样品信息页面，执行领样操作。

注意：样品领用时，由样品管理员登录,不是领用人

  • 打印样品标签 打印单个样品的样品标签，可选择打印正样、副样和备样。

    操作步骤如下： 点击当前样品右端【操作】-->【打印条码】，弹出打印样品标签操作页面，执行打印样品标签操作。

领样

功能概述

制样完成后，由检测人员向制样人员领取样品。

操作说明

领样品信息页面，如下图所示：

在领样品信息页面中执行领样操作。操作步骤如下：

步骤描述备注是否必须
第一步选择领样人√
第二步选择领样类型√
第三步勾选可领样品和检测项目√
第四步如果待领，则需要勾选“我要待领”，同时选择待领人，输入待领人用户密码×
第五步如果是本人领样，只需输入领样人用户密码×
第六步点击【领取样品】完成领样操作√

操作说明

打印样品标签操作页面，如下图所示：

在打印样品标签操作页面中执行打印样品标签，操作步骤如下：

计量样品

功能概述

计量样品 就是建立计量器具台账，查看计量器具的基本信息、流转明细、附件明细以及生成计量器具交接记录单。

操作页面如下图所示：

操作说明

  • 查看计量器具信息 在计量器具台账中，点击【查看】弹出当前计量器具的详细信息界面。界面如下图所示：

  • 计量器具查询 可根据器具状态、当前位置和部室类别进行精确查询或者模糊查询。另外委托单号部分可以根据用户自己的需求选择需要查询的内容，支持按照委托单号、交接单位、证书/报告单位、当前地点、收样地点和领取地点查询。
  • 查看子样信息 在计量器具台账界面中，点击【>】可以显示出该条记录下的子样信息，在子样信息中继续点击【>】可以查看该条子样信息的流转记录。如下图所示：

在计量器具子样及其流转记录信息界面中，点击【附件明细】，则显示当前子样附件记录信息。

  • 生成交接单 操作步骤如下：

第一步 选择计量器具 勾选需要交接的计量器具

第二步 填写交接单信息 点击【生成交接单】，弹出生成交接单对话框，如下图所示：

第三步 生成交接单 在生成交接单对话框中填写完毕，点击【确定】生成样品交接记录单。

样品流转

样品流转 就是对样品流转过程各个交接环节及其交接时生成的交接单进行管理。

样品交接单列表

功能概述

样品交接单列表 就是实验室的检测样品交接单的列表。一条记录对应一张交接单，对实验室的检测样品交接单进行管理添加样品交接单信息和查看样品交接单信息。操作页面如下图所示：

操作说明

  • 检索样品交接单信息 可根据交接单号、交接人、交样日期、接样人、接样日期时间段进行精确或者模糊查询。

  • 添加样品交接单信息 点击【添加样品交接单】，弹出样品交接操作页面，如下图所示：

第一步 在样品交接操作页面中，在条码号录入框中，可直接扫描条码、委托单号、交接单号条形码，系统会自动识别。扫描样品编码时自动显示样品子样信息；扫描委托单编号或者交接单编号时，需要一次将该单号之下的所有样品子样信息显示出来，可以通过扫样品子样编码逐条打勾，也可以手工批量勾选确认。

提醒：样品子样编号与委托单和交接单编号不能混合处理，即单样处理模式和单据（委托单和交接单）处理模式是互拆的：在单样处理模式中如果扫码单据编号择需提醒用户“单样处理模式中不能扫码单据编号，请确认！” ；同样，在单据处理模式中如果扫描不属于当前单据的样品编码需要提醒用户“样品XXX不属于当前单据，请确认。”

第二步 点击【批量接受】完成样品交接操作。

  • 查看样品交接单信息 点击【编辑】，弹出查看样品交接单信息操作页面，如下图所示：

样品上架

功能概述

对样品进行入库后的管理，并登记相关信息，包括存放位置等.对于需要变更保存位置（如阳性样品等），可记录变更信息，并可追溯到原来的保存信息。

操作说明

在主界面中点击【样品管理】-->【样品上架】菜单，弹出样品上架操作页面,如下图所示：

具体操作如下：

第一步 首先扫描样品管理员工牌号确定操作人。

第二步 扫描货位号码确定样品需要上架的货架号。

第三步 扫描样品子样编号、样品编号或样品所属委托单编号或样品所属交接单编号即可完成样品上架操作。

提醒：在条码号录入框中，可直接扫描样品或子样条码、委托单号、交接单号等信息，系统会根据不同的编码规则自动识别。系统自动记录当前（最后一次扫描结果）样品管理员、货位信息到流转记录中

注意：如果有多批样品需要上架，则只需重复第三步即可；如果需要更换货位，重复操作第二步；如果需要更换操作人，重复操作第一步

样品处置

功能概述

  • 样品完成检验后，系统实现样品剩余数量及其处理方式（消耗、销毁、移交）的记录和查询，支持批处理。
  • 对过了保存期、不需在保存的样品，可提醒管理人员进行处置。

操作说明

在主界面中点击【样品管理】-->【样品处置】菜单，弹出样品处置操作页面,如下图所示：

  • 查看检测样品信息 在样品处置操作页面中，点击【查看】弹出当前检测样品的详细信息界面。界面如下图所示：

  • 检测样品查询 可根据样品状态、当前位置和部室类别进行精确查询或者模糊查询。另外委托单号部分可以根据用户自己的需求选择需要查询的内容，支持按照委托单号、交接单位、证书/报告单位、当前地点、收样地点和领取地点查询。

  • 批量处置 就是针对需要处置的样品进行样品处置操作，平台将样品状态更新为“已处置”状态。至此，样品的整个生命周期结束。具体操作步骤如下：

第一步 选择需要处置的样品

第二步 点击【批量处置】，弹出批量处置对话框，如下图所示：

在批量处置对话框中选择处置方式，点击【确定】弹出样品处置操作成功提示对话框，点击【OK】完成样品处置操作。

  • 批量领取样品/器具

第一步 选择需要领取的样品

第二步 点击【批量领取样品/器具】，弹出批量领取对话框，点击【确定】完成样品领取操作。

  • 批量领取报告证书

第一步 选择需要领取报告的样品

第二步 点击【批量领取报告证书】，弹出批量领取对话框，点击【确定】完成样品报告证书领取操作。

场所管理

《检测和校准实验室能力通用要求》4.1.3条规定：“实验室的管理体系应覆盖在固定设施内、离开其固定设施的场所，或在相关的临时或移动设施中进行的工作。”为满足其规定，必须对实验室的多检测场所工作进行管理。

场所列表

功能概述

场所列表 就是实验室的检测场所列表。对实验室的检测场所进行管理添加场所信息、删除场所信息和编辑场所信息。操作页面如下图所示：

操作说明

  • 添加场所信息 点击【添加场所】，弹出场所信息编辑页面，执行场所信息编辑操作 。

  • 编辑场所信息 点击【编辑】，弹出场所信息编辑页面，执行场所信息编辑操作 。

  • 删除场所信息 点击【删除】，弹出删除确认对话框，点击【确认】完成删除。

货位管理

货位列表

功能概述

货位列表 就是实验室的样品货位列表。对实验室的样品货位进行管理，操作包括添加货位信息、删除货位信息和编辑货位信息。操作页面如下图所示：

在货位信息编辑页面中，分为左右两个区域。左边是组织结构选择区，右边是货位信息维护区。 在组织结构选择区选择部门，在货位信息维护区则显示选择部门的样品货位，并可对其进行增删改等维护工作。
