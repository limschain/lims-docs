人员管理

人员列表

功能概述

人员列表对实验室人员基本信息、系统登录用户相关信息、分配的岗位进行管理。操作包括员工入职、员工离职、更改信息和详情。操作页面如下图所示：

操作说明

  • 员工入职 员工入职，就是添加员工信息。具体操作如下：点击【员工入职】，弹出人员信息编辑页面，执行人员信息编辑操作 。

  • 更改信息 就是修改或变更员工信息。具体操作如下：点击【更改信息】，弹出人员信息编辑页面，执行人员信息编辑操作。

  • 详情 就是显示分配给当前员工的岗位信息。 具体操作如下：将鼠标悬停在【详情】，系统自动显示分配给当前员工的岗位列表。

  • 员工离职 就是将员工从平台中删除。具体操作如下：点击【员工离职】，弹出删除确认对话框，如下图所示：

在删除确认对话框中点击【确认】，完成员工离职操作。
